# HEALSON - איתור שירותים ורופאים (פנימי)

מערכת ווב פנימית עבור עובדי "הילסון" לאיתור מהיר של שירותים רפואיים מתוך Google Sheets.

## אפיון פונקציונלי
- **מקור נתונים:** Google Sheets (Read-Only).
- **חיפוש:** חופשי (שם/קוד שירות) + מסנני מחלקה וקטגוריה.
- **סינון קופות:** הצגת תוצאות רק עבור קופות עם ערך קיים (לפי עמודות ייעודיות).
- **תצוגה:** טבלה מרכזית + כרטיס רופא מורחב (Modal).
- **תיעוד (Logging):** רישום כל פעולת חיפוש בגיליון `Search_Logs`.

## מבנה נתונים (עמודות גוגל שיטס)
קוד פריט, שם פריט, צוות, קבוצת יומן, קטגוריית פו, סניף, מחלקה, פונה עצמית, החזר ופרטי, מחיר (פרטי), ועמודות קופות החולים.
